## Code

### Inline code

You can print to the terminal using command `puts: "hello, world!"`

### Mutli line code

#### Without highlighting

```
def hello_  world
    puts "Hello world"
end
```

#### With highlighting

```rb
def hello_  world
    puts "Hello world"
end
```
